# LightQuant 项目文档

欢迎使用 LightQuant 文档中心！本文档旨在帮助您快速了解、安装和使用 LightQuant 轻量级量化投资框架。

## 项目简介

LightQuant 是一个专为股票市场预测设计的轻量级量化投资框架。该框架整合了股票价格数据和金融新闻文本数据，支持多模态数据分析、模型训练和策略回测。框架采用模块化设计，提供了灵活的数据处理能力，能够简化和加速股票市场预测模型的研究与开发过程。LightQuant 集成了多种主流深度学习模型，包括 LSTM、BiLSTM、ALSTM、Adv-LSTM、DTML、SCINet、StockNet、HAN 和 PEN 等，可满足不同的研究需求。此外，项目还提供了完整的数据收集、处理和 LLM 因子提取流程，为研究者提供了一站式解决方案。

## 主要特性

LightQuant 框架具有以下核心优势：首先，框架支持多模态数据融合，能够同时利用股票价格数据和新闻文本信息进行联合预测，捕捉市场中的价格模式和消息面影响。其次，框架提供了丰富的预置模型选择，从基础的 LSTM 到复杂的注意力机制模型，满足不同复杂度的预测需求。第三，框架内置了完整的回测系统，支持多种交易策略的模拟和评估，帮助用户验证策略的有效性。第四，框架提供了灵活的配置接口，用户可以通过命令行参数或配置文件轻松调整模型和数据处理的各种设置。最后，框架支持 GPU 加速训练，能够高效处理大规模数据集和复杂模型。

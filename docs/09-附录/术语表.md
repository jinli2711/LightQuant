# 术语表

本术语表整理了 LightQuant 项目及相关领域常用的专业术语及其定义，帮助用户理解项目文档中使用的专业词汇。

## 通用术语

### 量化投资（Quantitative Investment）

量化投资是一种利用数学模型和计算机程序进行投资决策的方法。与传统的主观投资不同，量化投资基于数据分析和算法交易，追求投资的系统性和纪律性。量化投资的核心是发现市场中的统计规律，并基于这些规律设计交易策略。LightQuant 正是这样一个量化投资研究框架。

### 回测（Backtesting）

回测是使用历史数据模拟交易策略执行的过程。通过将策略的预测结果转化为交易信号，在历史数据上模拟执行，可以评估策略在历史市场环境下的表现。回测是量化投资研究中的核心方法，但需要注意历史表现不代表未来收益。

### 多模态（Multimodal）

多模态指同时使用多种类型的数据进行分析。在 LightQuant 中，多模态通常指同时使用价格数据和新闻文本数据进行股票预测。多模态方法能够利用更丰富的信息，通常能获得更好的预测效果。

## 数据相关术语

### OHLC

OHLC 是 Open（开盘价）、High（最高价）、Low（最低价）、Close（收盘价）的缩写，是股票价格数据的四个基本元素。这四个价格反映了一个交易周期内的价格波动情况，是技术分析的基础数据。

### 调整后收盘价（Adjusted Close）

调整后收盘价是考虑了分红、送股、拆股等因素调整后的收盘价。调整后收盘价能够更准确地反映股票的真实回报，是长期分析的重要指标。

### 标签（Label）

标签是监督学习中的目标变量，表示我们希望预测的内容。在 LightQuant 的股票预测任务中，标签定义为：如果下一个交易日收盘价高于当前交易日，则标签为 1（上涨）；否则标签为 0（下跌或持平）。

### 回看窗口（Look Back Window）

回看窗口是模型用于预测的历史数据长度。例如，look_back_window = 7 表示模型使用过去 7 天的数据来预测下一天的涨跌。回看窗口的选择影响模型能够捕捉的时间范围。

### 嵌入（Embedding）

嵌入是将离散的符号（如词语）映射到连续向量空间的技术。在 LightQuant 中，新闻文本通过嵌入层转换为向量表示，便于神经网络处理。好的嵌入能够保留词语的语义信息。

## 模型相关术语

### LSTM

长短期记忆网络（Long Short-Term Memory）是一种特殊的循环神经网络，能够学习长期依赖关系。LSTM 通过引入门控机制解决了传统 RNN 的梯度消失问题，是序列预测任务中最常用的模型之一。

### BiLSTM

双向 LSTM（Bidirectional LSTM）由两个方向的 LSTM 组成，分别处理正向和逆向的序列信息。BiLSTM 能够同时利用过去和未来的上下文信息，在许多任务上优于单向 LSTM。

### 注意力机制（Attention Mechanism）

注意力机制允许模型在处理输入时关注最相关的部分，而不是平等对待所有输入。在序列预测中，注意力机制使模型能够关注输入序列中最重要的时间步。

### 对抗训练（Adversarial Training）

对抗训练是一种通过在输入上添加对抗扰动来增强模型鲁棒性的技术。对抗训练使模型对输入的微小扰动更加不敏感，从而提升泛化能力。

### 预训练模型（Pretrained Model）

预训练模型是在大规模数据集上预先训练好的模型权重。使用预训练模型可以利用大规模数据学习到的知识，通常能够提升下游任务的效果。FinBERT 是专门在金融语料上预训练的 BERT 模型。

## 评估指标术语

### 准确率（Accuracy，ACC）

准确率是分类正确的样本占总样本的比例。对于二分类问题，准确率 = (真正例 + 真负例) / 总样本数。准确率是最直观的分类评估指标，但在样本不平衡时可能产生误导。

### 马修斯相关系数（MCC）

马修斯相关系数（Matthews Correlation Coefficient）是一个更稳健的二分类评估指标，考虑了所有四个预测类别的混淆矩阵情况。MCC 的取值范围为 -1 到 +1，+1 表示完美预测，0 表示随机预测。

### 年化收益率（ARR）

年化收益率（Annualized Return Rate）将策略的总收益转化为等效的年化收益率，便于不同时间长度的策略比较。计算公式为：ARR = (最终资产 / 初始资产)^(252/交易日天数) - 1。

### 夏普比率（SR）

夏普比率（Sharpe Ratio）衡量策略的风险调整后收益，计算每承担一单位风险所获得的超额收益。计算公式为：SR = (策略年化收益率 - 无风险利率) / 策略年化波动率。夏普比率越高越好。

### 最大回撤（MDD）

最大回撤（Maximum Drawdown）衡量策略在最坏情况下从峰值到谷值的最大跌幅，反映策略的极端风险。MDD 是风险控制的重要指标，MDD 越大表示策略可能遭受的最大损失越大。

### Calmar 比率

Calmar 比率将年化收益率与最大回撤结合，衡量每单位最大回撤获得的收益。计算公式为：CR = ARR / MDD。Calmar 比率越高表示风险收益特征越好。

### 信息比率（IR）

信息比率（Information Ratio）衡量策略相对于基准的超额收益及其稳定性。计算公式为：IR = 超额收益均值 / 超额收益标准差。IR 越大表示策略的相对表现越好且越稳定。

## 交易相关术语

### 做空（Short Selling）

做空是预期资产价格下跌时先卖出再买入的获利方式。投资者从券商借入股票卖出，等价格下跌后买入归还券商，赚取差价。LightQuant 默认策略支持做空。

### 交易成本（Transaction Cost）

交易成本是执行交易时产生的费用，包括佣金、印花税、过户费等。在回测中考虑交易成本可以获得更现实的策略评估结果。

### 滑点（Slippage）

滑点是预期成交价与实际成交价之间的差异，通常由于市场流动性和订单大小导致。滑点在高频交易和大额交易中尤为显著。

### 资产净值（Asset Value）

资产净值是投资组合的总价值，包括现金和持仓市值。回测中通过跟踪资产净值的变化来评估策略表现。

## 深度学习相关术语

### 批归一化（Batch Normalization）

批归一化是一种用于深度神经网络的正则化技术，通过在每个批次上标准化激活值来加速训练和提高稳定性。

### Dropout

Dropout 是一种正则化技术，通过在训练过程中随机丢弃部分神经元来防止过拟合。Dropout 使网络不依赖于特定的神经元，增强泛化能力。

### 梯度累积（Gradient Accumulation）

梯度累积是一种在有限显存下模拟大批量训练的技术。通过在多个小批次上累积梯度，然后统一更新模型参数。

### 混合精度训练（Mixed Precision Training）

混合精度训练使用 FP16 和 FP16 两种精度进行计算，可以减少显存占用和加速训练。混合精度训练需要 GPU 支持 FP16 计算。

## 项目特定术语

### CSMD

CSMD（Curated Multimodal Dataset）是 LightQuant 项目配套的多模态数据集，包含股票价格数据和金融新闻文本数据。CSMD50 包含 50 只股票，CSMD300 包含 300 只股票。

### SwanLab

SwanLab 是一个机器学习实验管理平台，用于跟踪实验、记录指标、可视化结果。LightQuant 使用 SwanLab 进行训练过程监控和实验记录。

### 单模态（Single Modality）

单模态指仅使用一种类型的数据。在 LightQuant 中，单模态通常指仅使用价格数据进行预测，不使用新闻数据。

### 多模态（Multimodal）

多模态指同时使用多种类型的数据。在 LightQuant 中，多模态通常指同时使用价格数据和新闻数据进行联合预测。
